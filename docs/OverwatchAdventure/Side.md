# 《守望大冒险》支线任务攻略
##### <Badge type="tip" text="发行版" />
### 1.孤雪人呢？
:::details 点击展开
接取地点：萌晗镇家门口  
任务攻略：和主线同时进行就可以完成，要注意在主线完成对孤雪刘念家的调查后，此支线会变成不可接的状态
:::

### 2.猫改不了吃鱼
:::details 点击展开
接取地点：皇心林下方营地处  
任务攻略：去上方钓鱼点
![猫改不了吃鱼](/pictures/OverwatchAdventure/猫改不了吃鱼.webp)
钓三条狗鱼回来给飞天猫。
:::

### 3.鬼屋测试
:::details 点击展开
接取地点：皇心镇不理吉他家左上方的骷髅面具男  
任务攻略：开鬼后躲到
![鬼屋测试](/pictures/OverwatchAdventure/鬼屋测试.webp)
这个柜子里，要注意的是只有第三个布帘后面没有尖叫陷阱。
:::

### 4.长枪和摩西
:::details 点击展开
接取地点：皇心镇民宅2 1F偷了洗澡男的衣服后上2F发现地上情书  
任务攻略：去找城门口最左边的这个人就行。
![长枪和摩西](/pictures/OverwatchAdventure/长枪和摩西.webp)
:::

### 5.遗书
:::details 点击展开
接取地点：皇心镇民宅3，注意别问他是不是开挂了  
任务攻略：皇心洞这个地方![遗书](/pictures/OverwatchAdventure/遗书.webp)开了，进去要打架。回来之后告诉他真相，后续还能在结冰村的酒馆和国王大道城的银碟游乐场见到他。会直接送100积分，特别省银碟的时间。
:::

### 6.药草
:::details 点击展开
接取地点：皇心城堡3F的医务室    
任务攻略：非常非常简单，就是去皇心林找地上发光的东西。
:::

### 7.破冰达人
:::details 点击展开
接取地点：第一次出海时，地图右上角的结冰村最上面的房子  
任务攻略：去KK家后去JJ家摸到4个线索记住，再回去找KK答题就可以了。
:::

### 8.恋与守望
:::details 点击展开
接取地点：国王大道城左下角的甜品店里的小女孩  
任务攻略：在守望中专教室3的桌子上找到任务物品。
:::

### 9.赛博精神病
:::details 点击展开
接取地点：国王大道城右上角要斜着走楼梯的2楼平台  
任务攻略：在街上遇到这个人![赛博精神病](/pictures/OverwatchAdventure/赛博精神病.webp)，打败就行了。
:::

### 10.蟹特！我蟹呢！
:::details 点击展开
接取地点：国王大道城公寓-蟹の家  
任务攻略：螃蟹在这些地方![蟹特！我蟹呢！](/pictures/OverwatchAdventure/蟹特！我蟹呢！.webp)，注意画黄圈的螃蟹会触发战斗。
:::

### 11.童话世界
:::details 点击展开
接取地点：第二次出海时，地图上面的甜蜜镇的精品女装店
任务攻略：调查后面的柜子，找出雾子的5个皮肤。其实可以全选【拿】，因为错了的话KYO会说。
:::

### 12.神偷李武
:::details 点击展开
接取地点：里阿尔托城河神行政大楼 2F 下到的监狱里。  
任务攻略：在被大祭司追杀的时候就能去监狱里救人了。
:::

### 13.灭鼠行动
:::details 点击展开
接取地点：里阿尔托城民宅5  
任务攻略：最好在里阿尔托城换一下装备，提升一下实力再去打，是要打仓鼠、地鼠、狂鼠，略难打。
:::

### 14.代练和老板
:::details 点击展开
接取地点：里阿尔托城河神行政大楼1F 任务板  
任务攻略：去别墅3的2F可以找到代练。推荐至少一个人搞到最好的装备或等级＞13再去打，是较有难度的战斗。
:::
