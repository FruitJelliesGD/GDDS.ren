## 异常状态
### 中毒：每回合持续掉血![中毒](/pictures/icons/中毒.webp)
### 燃烧：每回合持续扣血![燃烧](/pictures/icons/燃烧.webp) 
### 感受痛苦：每回合持续掉大量血![感受痛苦](/pictures/icons/感受痛苦.webp) 
### 愤怒：命中率下降、攻击和魔攻增强![愤怒](/pictures/icons/愤怒.webp) 
### 魅惑：攻击己方![魅惑](/pictures/icons/魅惑.webp) 
### 被侵入：不可使用技能和大招![被侵入](/pictures/icons/被侵入.webp) 
### 反弹：将受到伤害等量反弹给对方![反弹](/pictures/icons/反弹.webp) 
### 睡眠：不可行动，被攻击后解除状态![睡眠](/pictures/icons/睡眠.webp) 
### 昏迷：不可行动，被攻击后不解除状态![昏迷](/pictures/icons/昏迷.webp) 
### 马枪：命中率下降![马枪](/pictures/icons/马枪.webp) 
### 矩阵怼脸：下一次伤害为0![矩阵怼脸](/pictures/icons/矩阵怼脸.webp) 
### 暴击增加：暴击率提高![暴击增加](/pictures/icons/暴击增加.webp) 
### 毛妹盾：无敌且解除大部分负面状态，持续2回合。![毛妹盾](/pictures/icons/圣or毛妹盾.webp) 
### 圣：无敌且解除大部分负面状态，持续5回合。![圣](/pictures/icons/圣or毛妹盾.webp) 
### 无法战斗：死了，用技能或者道具复活吧![无法战斗](/pictures/icons/无法战斗.webp) 

## 属性介绍
:::code-group
```[攻击]
决定自由人全部伤害、重装一半的伤害
```
```[魔攻]
决定辅助全部伤害和治疗量、重装一半的伤害
```
```[防御]
决定受到敌人攻击技能时，可以抵消的伤害量
```
```[魔防]
决定受到敌人魔攻技能时，可以抵消的伤害量
```
```[最大HP]
决定HP总量
```
```[最大MP]
决定MP总量
```
```[敏捷]
决定攻击顺序
```
```[幸运]
决定异常状态发生的概率
```
```[TP]
每次战斗后刷新，可以在战斗内释放大招，受击和释放技能都会增加TP点数
```
:::
