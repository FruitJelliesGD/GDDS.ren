## 属性介绍
:::code-group
```[攻击]
决定自由人全部伤害、重装一半的伤害
```
```[魔攻]
决定辅助全部伤害和治疗量、重装一半的伤害
```
```[防御]
决定受到敌人攻击技能时，可以抵消的伤害量
```
```[魔防]
决定受到敌人魔攻技能时，可以抵消的伤害量
```
```[最大HP]
决定HP总量【HP为0会死亡】
```
```[最大MP]
决定MP总量【MP不足时无法使用技能】
```
```[敏捷]
决定攻击顺序
```
```[幸运]
决定异常状态发生的概率
```
```[TP]
每次战斗后刷新，可以在战斗内释放大招，受击和释放技能都会增加TP点数
```
:::
